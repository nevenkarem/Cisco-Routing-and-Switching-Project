# Cisco Routing and Switching Project

## Project Overview
This project demonstrates the implementation of core CCNA networking technologies using Cisco Packet Tracer. The lab focuses on network segmentation, routing, security, and switch management.

## Technologies Implemented
- VLAN Configuration (VLAN 2, 3, 4, 5)
- Inter-VLAN Routing
- DHCP
- Access Control Lists (ACLs)
- EtherChannel
- PVST+
- Port Security
- Telnet Remote Access

## Network Topology
The network consists of multiple VLANs connected through routing between VLANs. Security and management features were configured to provide controlled access and efficient network operation.

## Configuration Tasks

### VLANs
- Created VLAN 2, VLAN 3, VLAN 4, and VLAN 5
- Assigned switch ports to the appropriate VLANs

### Inter-VLAN Routing
- Configured routing between VLANs to enable communication across different networks

### DHCP
- Configured automatic IP address assignment for client devices

### ACLs
- Implemented Access Control Lists to control network traffic between VLANs

### EtherChannel
- Configured EtherChannel to provide link redundancy and increased bandwidth

### PVST+
- Implemented Per-VLAN Spanning Tree Protocol (PVST+) to prevent switching loops

### Port Security
- Configured port security to restrict unauthorized devices

### Telnet Remote Access
- Configured Telnet for remote switch management
